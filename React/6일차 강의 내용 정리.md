### 단락회로 평가

> 왼쪽에서 오른쪽으로 연산하게 되는 논리연산자의 연산 순서를 이용함
> true, false 뿐만 아니라 truthy, falsy 도 피연산자로 이용할 수 있음

```javascript
// AND 연산자
console.log(true && true);
// OR 연산자
console.log(true || false);
// NOT 연산자
console.log(!true);
```
