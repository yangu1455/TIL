# 예제 02. 기초 함수


##### 💡 본 문제는 함수를 직접 정의하고 호출하는 문제입니다.



> 가로와 세로의 길이를 각각 a, b로 받아 사각형 넓이와 둘레를 함께 반환하는 함수 rectangle을 정의하시오. 
사각형이 가로가 20, 세로가 30일 때의 결과를 출력하시오.

* 사각형 넓이 : 가로 * 세로 
* 사각형 둘레 : (가로 + 세로) * 2
>
### Output
---

```
(600, 100)
```



## 작성한 코드
----

```python
# 예제 2

def rectangle(x, y):
    return x * y, 2 * ( x + y )

print(rectangle(20, 30))
```
